# Lex3 账户托管系统

## 功能说明

用户登录平台后，系统自动保存Cookie，之后可自动发布。

## 支持平台

| 平台 | 状态 | 账号 |
|------|------|------|
| 知乎 | ✅ 已托管 | 徐子 |
| 公众号 | ⏳ 待登录 |
| 36氪 | ⏳ 待登录 |
| 法律快车 | ⏳ 待登录 |

## 使用方法

### 1. 账户托管

访问各平台登录页面，登录后点击「托管此账户」

### 2. 自动发布

选择内容 → 选择平台 → 点击发布

### 3. 发布日志

记录每次发布结果

## 技术实现

### Cookie存储

```javascript
// 保存Cookie
async function saveCookies(page) {
    const client = await page.target().createCDPSession();
    const cookies = await client.send('Network.getAllCookies');
    localStorage.setItem('zhihu_cookies', JSON.stringify(cookies.cookies));
}

// 加载Cookie
async function loadCookies(page) {
    const cookies = JSON.parse(localStorage.getItem('zhihu_cookies'));
    await page.setCookie(...cookies);
}
```

### 自动发布流程

1. 打开平台页面
2. 加载保存的Cookie
3. 检查登录状态
4. 发布内容
