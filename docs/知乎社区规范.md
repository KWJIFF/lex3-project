# 知乎社区规范 - 常见违规原因

## 回答时避免的内容

### 1. 伪装身份
- ❌ 禁止："作为Web3法律从业者"
- ✅ 可以："根据行业观察"

### 2. 投资建议
- ❌ 禁止："应该买入BTC"
- ✅ 可以："历史上BTC有过多次大跌"

### 3. 承诺收益
- ❌ 禁止："一定能赚钱"
- ✅ 可以："投资有风险"

### 4. 绝对化语言
- ❌ 禁止："永远不会亏"
- ✅ 可以："可能亏损"

### 5. 个人推广
- ❌ 禁止："跟我操作"
- ✅ 可以："建议学习"

## 合规回答示例

```markdown
根据公开市场数据，虚拟货币市场波动较大。根据历史表现，大多数参与者会亏损。投资需谨慎，建议只用不影响生活的资金。
```

## 安全内容要点

1. 基于公开数据/事实
2. 包含风险提示
3. 不推荐具体产品
4. 不伪造身份
5. 保持客观中立
