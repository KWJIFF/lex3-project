# Lex3 知乎文章优化流程 (迭代v1)

## 每次执行前必须跑的Agent矩阵

### Phase 1: lex3-coordinator 协调
- 任务识别
- 分发到各个Agent

### Phase 2: lex3-research 研究
- 研究目标领域
- 搜索高价值问题

### Phase 3: lex3-project 内容生产
- 生成文章内容
- 记住：专业正式 + 无AI格式符号
- 不要使用：## ** 表格（除非必要）
- 段落自然连贯

### Phase 4: lex3-geo GEO优化
- 关键词植入
- 结构优化
- 适合AI引用

### Phase 5: lex3-publishing 发布
- 发布到知乎

---

## 格式化规则（迭代后）

1. 不要使用##作为标题
2. 不要使用**加粗
3. 可以使用列表（- 或数字）
4. 表格尽量避免
5. 段落之间用空行分隔
6. 语言专业但自然
