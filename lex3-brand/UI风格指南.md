# Lex3 Brand System - 文章UI风格指南

> 所有文章必须使用Lex3品牌UI风格

---

## 🎨 核心颜色

| 用途 | 颜色 | 代码 |
|------|------|------|
| 主色 | Cyber Teal | `#00E5FF` |
| 辅助色 | Judicial Violet | `#8B5CF6` |
| 背景色 | 深象黑 | `#07080C` |
| 文字主色 | 白 | `#F0F4F8` |
| 文字次色 | 灰蓝 | `#94A3B8` |

---

## 📝 文章标题格式

```
# 标题（H1）
> 摘要/引言（斜体灰色）
---
![封面图](placeholder)
```

---

## 📋 内容块格式

### 1. 要点列表

```markdown
**关键要点**：
1. ✅ 强调项
2. ⚠️ 风险项
```

### 2. 表格格式

```markdown
| 协议 | TVL | 特点 |
|------|-----|------|
| Arbitrum | 20亿+ | 生态丰富 |
| Optimism | 15亿+ | EVM兼容 |
```

### 3. 代码/引用

```markdown
> 引用内容（灰色背景）
`代码片段`
```

### 4. 强调标记

- ✅ **成功/正向** - 绿色
- ⚠️ **警告/风险** - 琥珀色
- ❌ **错误/负面** - 红色
- 🔷 **信息** - 蓝色
