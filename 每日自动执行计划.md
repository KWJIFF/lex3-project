## 🎯 社区互动任务（每日）

### 阶段1：加入圈子
- [ ] 访问知乎圈子页面 (https://www.zhihu.com/ring)
- [ ] 筛选相关圈子（区块链、Web3、加密货币、RWA、NFT、DeFi、稳定币、投资理财、职业成长）
- [ ] 优先加入影响力大的圈子（成员数≥1000）
- [ ] 一键加入所有相关圈子
- [ ] 记录已加入的圈子列表

### 阶段2：每日签到打卡
- [ ] 进入每个加入的圈子
- [ ] 点击签到按钮完成每日签到
- [ ] 记录签到结果

### 阶段3：互动参与
- [ ] 浏览圈子中的热门内容
- [ ] 点赞高质量内容（≥10个/天）
- [ ] 评论专业内容（≥5条/天，≥100字/条）
- [ ] 发布想法到相关圈子（≥3条/天）

### 阶段4：Agent矩阵质量检查
- [ ] **每次发布内容前**：运行lex3-agent-matrix进行质量检查
- [ ] 验证内容专业性和准确性
- [ ] 优化SEO和GEO
- [ ] 检查合规性（避免违规内容）
- [ ] 确认风险提示（投资内容标注"投资有风险，入市需谨慎"；法律内容标注"仅供普法参考"）
