# Lex3 自媒体托管平台

## 功能定位

类似"蚁小二"的多平台自媒体账号托管系统：
- 多平台账号管理
- 定时发布内容
- 数据统计汇总
- 账号授权管理

---

## 平台架构

```
┌─────────────────────────────────────────────────────────┐
│              Lex3 Publishing Hub                        │
│                  自媒体托管平台                          │
├─────────────────────────────────────────────────────────┤
│  账号管理    内容库    发布管理    数据中心    设置     │
└─────────────────────────────────────────────────────────┘
```

## 核心功能模块

### 1. 账号管理

```
功能：
- 添加账号（扫码/授权）
- 账号状态监控
- 账号分类管理
- 账号信息编辑

支持方式：
- 公众号：管理员扫码授权
- 小红书：扫码授权
- 知乎：扫码授权
- 抖音：扫码授权
```

### 2. 内容库

```
功能：
- 文章/笔记/视频存储
- 内容模板
- 定时发布设置
- 内容预览
```

### 3. 发布管理

```
功能：
- 一键发布多平台
- 定时发布
- 发布状态追踪
- 发布失败重试
```

### 4. 数据中心

```
功能：
- 阅读量/播放量统计
- 粉丝增长趋势
- 互动数据（点赞/评论/收藏）
- 数据报表导出
```

## Agent集成

### lex3-project 调用

```
内容生产完成后：
→ lex3-project 调用 lex3-publishing
→ lex3-publishing 自动发布到多平台
→ 返回发布结果
```

## 集成到lex3-project（Node 11: 多平台发布）

原流程：
```
11.11.1 登录公众号后台手动发布
...
```

改进后：
```
11.11.1 调用lex3-publishing发布
11.11.2 一键发布到公众号+小红书+知乎+抖音+视频号
11.11.3 自动追踪发布状态
11.11.4 处理发布异常
11.11.5 返回发布结果
```
